# Top Games

A demo app to show a list of top games using `Twitch TV API`.  

**Requirements**  

- Swift 4
- Storyboard
- UICollectioView
- Endless Scroll
- Autolayout
- CoreData
- Pull to Refresh
- Unit Tests
- cocoapods

**Pods**

- KTCenterFlowLayout
- Alamofire
- AlamofireImage
- SwiftLint
- Quick
- Nimble
